# 📬 Maincrafts Contact Form

A full-stack Contact Form application built using Spring Boot, HTML, CSS, and JavaScript.  
It includes form validation, theme switching, and submission tracking with a backend API.

---

## 🚀 Features

- Contact form with client-side and server-side validation
- Light/Dark theme toggle
- Stores and tracks form submissions
- Responsive UI for mobile and desktop
- REST API backend using Spring Boot

---

## 🛠️ Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Java
- Spring Boot
